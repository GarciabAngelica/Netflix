# 🎬 Clon de Netflix

## 📌 Plataforma clonada

Se realizó un clon inspirado en Netflix, enfocándose principalmente en la interfaz visual tipo streaming platform.

El proyecto replica elementos característicos como:

- Navbar oscura con logo
- Tarjetas de contenido estilo catálogo
- Diseño responsive
- Efectos hover en las cards
- Estética moderna y minimalista

---

# 🛠️ Componentes Bootstrap utilizados

Durante el desarrollo se utilizaron varios componentes y utilidades de Bootstrap:

## Componentes

- Navbar
- Cards
- Flex utilities
- Containers

## Utilidades

- Márgenes y espaciados (`mt-`, `mb-`, `p-`)
- Responsive Design
- Flexbox
